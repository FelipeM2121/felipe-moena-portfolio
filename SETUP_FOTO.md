# 📸 Instrucciones para Agregar la Foto de Perfil

## Paso 1: Guardar la Foto

1. La foto que compartiste necesita ser guardada con el nombre **`profile.jpg`**
2. Debe colocarse en la carpeta: **`portfolio/public/images/`**

La estructura debe quedar así:
```
portfolio/
└── public/
    └── images/
        └── profile.jpg  ← Tu foto aquí
```

## Paso 2: Verificar la Foto

- Tamaño recomendado: 800x1000 px (o similar)
- Formato: JPG o PNG
- Asegúrate de que sea una foto clara y profesional

## Paso 3: Ejecutar el Proyecto

Una vez la foto esté en su lugar:

```bash
# 1. Navega a la carpeta del portfolio
cd portfolio

# 2. Instala las dependencias (primera vez)
npm install

# 3. Inicia el servidor de desarrollo
npm run dev
```

El portafolio se abrirá automáticamente en tu navegador y verás:
- Tu foto en la sección Hero (lado derecho)
- Tu nombre y descripción profesional
- Todas las secciones de experiencia, proyectos, etc.

## 🎯 Próximos Pasos

Una vez que confirmes que la foto está agregada:

1. ✅ Verifica que el sitio se vea correctamente
2. ✅ Navega por todas las secciones
3. ✅ Prueba el formulario de contacto
4. ✅ Verifica en móvil que sea responsive
5. 🚀 Deploy a producción (Netlify/Vercel)

## 💡 Ayuda Rápida

**Si no ves la foto:**
- Verifica el nombre del archivo (debe ser exactamente `profile.jpg`)
- Verifica la ruta: `portfolio/public/images/profile.jpg`
- Recarga el navegador (Ctrl+F5)

**Si necesitas cambiar la foto:**
- Solo reemplaza el archivo `profile.jpg` en la misma carpeta
- Recarga el navegador

---

**¡Tu portafolio está listo para ser personalizado! 🚀**
